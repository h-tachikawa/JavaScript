ng-repeat

より便利にループを処理するためループ内でインデックスが取得できます。
また、ng-repeat-start ng-repeat-endで、明示的にループ開始と終了を宣言することが可能です

シンプルな配列

  • {{ author }}

ハッシュオブジェクトの配列

  • ID: {{ author.id }} Name: {{ author.name }}

ハッシュオブジェクトのみ

  • {{ key }} - {{ val }}

インデックスを取得する

day $index $first $middle $last $even $odd
{{ day }} {{ $index }} {{ ($first)? '○': '×' }} {{ ($middle)? '○': '×' }} {{ ($last)? '○' : '×' }} {{ ($even)? '○' : 'x' }} {{ ($odd)? 'o' : 'x' }}

ng-repeat-start/ng-repeat-end

index : {{ $index }}
